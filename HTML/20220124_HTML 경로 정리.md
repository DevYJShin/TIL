# 절대 경로

> * "/Context Name"으로 시작
> * URL - http://Server IP:Port Number/Context Name/HTML File Save Path/HTML File Name
> * http://Server IP:Port Number까지 생략
> * 절대 경로는 경로 쓸 때 "/ " 로 시작 


# 상대 경로

> * 현재 Page가 보여지는 html 현재 디렉토리를 기준
> * 상위 경로 : ../
> * 같은 경로 : 생략가능 or ./
> * 하위 경로 : 폴더 지정  ex) a/a.html
